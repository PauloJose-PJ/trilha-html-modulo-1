O que são Clients

(Início)

Conceitos de Client: O cliente é o qual consome onde o procesor é fazer uma requisição para o Protocolo HTTP onde esse protocolo faz uma solicitação ao Servidor para voltar ao HTTP e ir DIRETO ao cliente.

Hospedagem: Hospedamos nossos sites que fazemos no próprio Servidor entre milhares que existem no mundo, escolhemos aquele mais próximo devido a latência, preço, suporte e lembrando eles ficam 24horas ligados que é o idéial para atenter a todos (digo todos em horario variado, pois se quiser algo maior terá que ter um suporte de espaço maior).

Conceitos de Servers

(Início)

Sobre Servidores:

Sobre Servidores: Serve de apoio a outras maquinas da rede, não sendo só para internet sendo assim podendo ser também local. Maquinas mais sofisticadas para grandes processamento ou pequenos. Servidores em conjunto são colocados em locais chamados datacenters, onde varias ou só uma empresa usa.

Hardwares e Softwares em Servidores Fazemos uma requisição HTTP para um Servidor Web e ele processa e volta como HTML, CSS, JS, Img, Texto.

Tipos de Servidores

  • Proxy - O exemplo abaixo é o Servidor Proxy que devido a configuração ele faz um bloqueio de alguns sites para um determinado ou todos Clientes independente se é celular, notebook, ou pc.
  • Firewall - É parecido com o Proxy mas para ambiente Externo quando algum aplicativo malicioso quem entrar no lado do Cliente.
  • Web Server - Tem o Apache e o WampServer onde você faz uma requisição igual na explicação anterior e volta conteúdos html, css, js e etc.
  • E-mail Server - Servidor voltado para e-mails
  • Web Server com Database - O Database esta sendo processada em outro servidor e não necessáriamente na Aplicação Web
  • Intranet - Disponível somente dentro de uma rede corporativa local
  • DNS - Serve para traduzir o Nome de um site (exemplo: www.gogle.com.br) para um IP (identidade de um site) e procurar numa lista e leva-lo para determinado local ou seja ele direciona o site que você quer navegar.
  • Hospedagem de Sites - http's' - Conexão segura de ponta a ponta. A empresa de Hospedagem pode ser servidor de email, ou streaming etc... para melhor tipo de hospedagem.
  • Registro - Para ter um registro você paga um valor anualmente para ter o nome do site e tem casos que uma pessoa pega esse nome e registra e você tem que comprar dela para ter o registro.
  • FTP - É uma conexão que você faz com o servidor onde você mandará arquivos
